## Base URL

```
https://csboard.com/v1
```

Every price in this API is **USD**. Use [`GET /v1/currency`](/api-reference/get-currency)
to convert to a local currency.

## Quick start

```bash
curl https://csboard.com/v1/listings \
  -H "Authorization: Bearer csb_pub_..."
```

Only [`GET /v1/health`](/api-reference/get-health) works without a key; every
other endpoint requires one. See [Authentication](/authentication).
