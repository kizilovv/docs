API CSBoard даёт вам программный доступ к маркетплейсу CS2-скинов. Получайте актуальные доступные для покупки листинги с float-значениями, paint seed, данными о стикерах и ценами в USD; скачивайте полный gzip-снапшот прайс-листа для массовой загрузки каталога; получайте курсы валют, согласованные с платёжными процессами сайта; и размещайте автоматические ордера на покупку с прямым списанием с баланса CSBoard — всё через единый REST API с корнем `https://csboard.com/v1`.

## Быстрый старт

Получите актуальные листинги одним запросом — настройка сводится к API-ключу:

```bash
curl https://csboard.com/v1/listings \
  -H "Authorization: Bearer csb_pub_..."
```
