CSBoard API 让您可以通过编程方式访问 CS2 皮肤市场。获取实时可购买挂单，包含 float 值、paint seed、贴纸数据和美元价格；拉取完整的 gzip 压缩价格列表快照，用于批量目录摄入；获取与站点支付流程一致的汇率；并下达自动化购买订单，直接从您的 CSBoard 余额中扣款——一切都通过位于 `https://csboard.com/v1` 的单一 REST API 完成。

## 快速开始

通过一次请求即可获取实时挂单——除了 API 密钥之外无需任何设置：

```bash
curl https://csboard.com/v1/listings \
  -H "Authorization: Bearer csb_pub_..."
```
